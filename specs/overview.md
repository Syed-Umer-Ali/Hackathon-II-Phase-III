# Project Overview: Evolution of Todo

## Purpose
The Evolution of Todo is a multi-phase hackathon project designed to demonstrate the transition from a simple CLI app to a sophisticated, cloud-native, AI-powered distributed system.

## Current Phase: Phase III - AI-Powered Todo Chatbot
We are currently in Phase III. The goal is to implement a conversational interface using natural language to manage tasks, powered by the OpenAI Agents SDK and the Model Context Protocol (MCP).

## Tech Stack (Phase III)
- **Frontend**: Next.js (Custom Chat UI)
- **Backend**: FastAPI
- **AI**: OpenAI Agents SDK
- **Protocol**: Official MCP SDK
- **Database**: Neon PostgreSQL (Serverless)
- **Auth**: Better Auth (JWT Integration)
